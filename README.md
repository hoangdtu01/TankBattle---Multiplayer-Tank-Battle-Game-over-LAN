\# Ring Duel



LAN Multiplayer Tank Battle Game developed using Java and LibGDX.



\## Features



\- Room Creation \& Join

\- Ready System

\- TCP Signaling Server

\- UDP Gameplay Networking

\- Host Authoritative Architecture

\- Client Prediction \& Reconciliation

\- Shield, Dash and Knockback Mechanics



\## Architecture



Client

&#x20;├── TCP Signaling

&#x20;└── UDP Gameplay



Server

&#x20;├── Room Management

&#x20;├── Match Start

&#x20;└── Player Synchronization



\## Technologies



\- Java

\- LibGDX

\- TCP Socket

\- UDP DatagramSocket
